### 操作说明
| 操作 | 按键 |
|------|------|
| 移动视角 | WASD |
| 缩放 | Q / E |
| 选中单位 | 左键单击 或 框选 |
| 移动单位 | 右键点击目标 |
| 建造建筑 | 右侧面板按钮 → 左键放置 → 右键取消 |
| 停止 | S |
