# RA2 科技时代 — 游戏设计文档

## 概述

基于红色警戒2的「科技时代」MOD 精神，打造一款跨平台 2D 即时战略游戏。保留经典 RTS 核心玩法，加入科技时代特色：中国阵营、进阶科技树、超级武器。

## 核心玩法

### 基础循环
```
采集资源 → 建造基地 → 生产军队 → 摧毁敌人
     ↑                              ↓
     └──── 占领更多矿源 ←── 扩张领地 ←┘
```

### 胜利条件
- 摧毁敌方所有建筑（征服模式）
- 可选：摧毁特定目标、限时得分

## 阵营设计

### 盟军（Allied）
- 特点：空中优势、高速机动
- 特色单位：光棱坦克、幻影坦克、夜鹰直升机
- 超级武器：天气控制器（闪电风暴）

### 苏联（Soviet）
- 特点：重装甲、火力压制
- 特色单位：天启坦克、基洛夫空艇、磁暴步兵
- 超级武器：核弹

### 中国（科技时代特色）★
- 特点：科技优势、电子战
- 特色单位：
  - 炎黄坦克（重型主战）
  - 黑客（瘫痪敌方建筑）
  - 电子干扰车（范围降低敌方命中率）
  - 轨道炮台（远程精确打击）
- 超级武器：
  - EMP 脉冲（范围瘫痪敌方载具和建筑）
  - 轨道炮击（卫星制导打击）

## 资源系统

- **资金（$）**: 采矿获得，用于建造和生产
- **电力**: 发电厂提供，建筑消耗。断电时生产效率降低 50%

## 科技树

```
建造场 → 发电厂
              ↓
           矿场
              ↓
           兵营 ──→ 作战实验室 ──→ 超级武器
              ↓                      ↓
         战车工厂 ──→ 空指部    进阶科技单位
```

### 科技时代特色科技层级

| 等级 | 需求 | 解锁 |
|------|------|------|
| T1 | 建造场 | 发电厂、矿场、兵营、围墙 |
| T2 | 兵营 + 矿场 | 战车工厂、防御塔 |
| T3 | 战车工厂 | 空指部、进阶单位 |
| T4 | 作战实验室 | 超级武器、顶级单位(e.g., 天启、光棱、炎黄) |

## 地图系统

- 2D Tilemap 地形
- 地形类型：草地、沙漠、水域、山脉（不可通行）、道路
- 矿源：可开采的金矿/矿石，分布在地图上
- 建筑用地：可放置建筑的平整地形

## 联机架构

- 传输层：Godot ENet (UDP)
- 模式：客户端-服务器
- 房间系统：创建/加入/退出
- 同步方式：命令帧同步（lockstep）
- 支持：局域网发现 + 直连 IP

## 操作方式

### 鼠标
- 左键：选中单位/建筑
- 左键拖拽：框选
- 右键：移动/攻击/确认位置
- Shift+左键：添加到选中

### 键盘
- 数字键 1-9：编队
- A：攻击移动
- S：停止
- Space：跳转到最近事件
- Tab：切换小地图模式

## 视觉风格

- 俯视 2D 像素/手绘风格
- 致敬原版红警2视觉
- 科技时代单位有更现代的外观

## 技术规格

- 帧率：60 FPS 目标
- 分辨率：1280×720 最低，支持任意缩放
- 最大单位数：单局 ~500 单位
- 网络延迟容忍：<200ms
