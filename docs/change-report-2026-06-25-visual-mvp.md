# 变更报告 — 2026-06-25 可视化层 + 可玩 MVP

## 概述

完成游戏可视化渲染层，实现可玩 MVP。玩家可以在地图上看到单位、建筑、战斗特效，框选操作，AI 自动对战。

## 核心变更

### 可视化渲染
所有游戏实体使用 `_draw()` 程序化渲染，无需外部素材：
- **步兵**: 阵营色圆形 + 选中绿环 + 血条 + 名称
- **载具**: 矩形车身 + 圆形炮塔 + 炮管指向移动方向
- **建筑**: 矩形色块 + 类型标记（电厂黄芯/兵营亮块/矿场金币）
- **框选**: 绿色半透明矩形
- **放置预览**: 绿色/红色（可/不可放置）
- **子弹轨迹**: 阵营色射线
- **死亡爆炸**: 橙色粒子扩散 + 实体缩小消失

### 新增系统 (3)
| 系统 | 文件 | 说明 |
|------|------|------|
| 小地图 | Minimap.gd | 200×200px 右下角地形/单位/视口框 |
| 爆炸特效 | ExplosionEffect.gd | 粒子动画系统 |
| 战斗视觉 | CombatVisuals.gd | 子弹轨迹线 |

### 重写系统 (6)
- Infantry.gd: standalone CharacterBody2D + _draw 圆形渲染
- Vehicle.gd: standalone CharacterBody2D + _draw 坦克渲染
- Building.gd: _draw 矩形渲染 + 类型标记
- BuildingManager.gd: 相机坐标修复 + _draw 放置预览
- UnitManager.gd: 相机坐标框选 + 散开队形 + 自动交战
- GameWorld.gd: 玩家初始部队部署 + 爆炸/视觉调度

### 新增配置
- export_presets.cfg: macOS + Windows 导出配置
- build.sh: 一键构建脚本
